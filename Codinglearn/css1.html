CSS, Was ist CSS

CSS ist die Style-Programmiersprache für HTML.

Anfang

Also, wenn ihr mit CSS programmiert, braucht ihr die HTML-Tags... Also, wenn ihr beispielsweise machen wollt, dass euer Text größer ist, gebt ihr das ein:

p {font-size: 23px;}

Diese speziellen Eigenschaften findet ihr hier:

  • p {font-size: 23px;} Größe
  • p {font-family: Arial, Helvetica, sans-serif;} Schrift
  • p {color: red;} Farbe
  • p {background-color: red;} Hintergrundfarbe (Ich empfehle: background-color: rgb(197, 236, 236);)
  • p {text-align: center;} (Zu empfehlen: center)

Diese Style elemente wie color.... kannst du auch beispielsweise bei einem h1 oder einem Link benutzen....
